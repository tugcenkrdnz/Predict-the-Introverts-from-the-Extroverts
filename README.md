# 🧠 Personality Type Predictor (ML Project)

Bu proje, bir kişinin günlük alışkanlıklarına bakarak **Introvert (İçe Dönük)** veya **Extrovert (Dışa Dönük)** olup olmadığını tahmin eden bir Makine Öğrenmesi modelidir.

## 🚀 Başarı Oranı
- **Model:** AdaBoost Classifier
- **Accuracy:** %96.86
- **Veri Seti:** Kaggle Personality Prediction Dataset

## 🛠️ Kullanılan Teknolojiler
- Python (Pandas, Scikit-learn)
- AdaBoost Classifier
- Streamlit (Arayüz)
- Joblib (Model Saklama)
